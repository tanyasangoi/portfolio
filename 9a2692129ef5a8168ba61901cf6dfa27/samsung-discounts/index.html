For this project, I worked with a team of designers to redesign the Discount Experience for the Samsung Discount Program.
I also revised the content strategy for customer communication.



Problem Background Process Solution Outcome

Problem

The current design on the Discount pages had a lot of distractions which resulted in a lot of people not signing up for the discount programs. There was also a lack of a consistent content strategy for communication with customers. Prioritizing content and removing distractions was necessary to optimize landing pages for conversions.

Background

This program offers discounts on Samsung products to employees, students, government officials, and military personnel. While each segment of the program targets a certain group of people, the overarching objective of this program was to motivate people to create a Samsung account and use the Samsung Rewards program as well. Thus there was a need to have a consistent message across the different segments.


Process

We performed a competitive analysis of some companies and understood the strategies the employed for discount programs. We also looked at existing user feedback to understand the breakpoints in the current system. This was followed by ideation, sketching, and refinement of the proposed ideas which were finally prototyped in high fidelity.

Solution

The goal of the redesign was to create a templated design system to:

  1. Streamline development and content creation
  2. Maintain brand identity
  3. Unify the experience across all entry points
Redesign the website experience

With the removal of promotional content at the top of the page, the user can quickly know that he is in the right place. The page informs the users of what the Discount Program offers and how to get the discount. Additional information about benefits and offers is below the main Call to Action button at the bottom of the page.


Redesign the customer communication

This was followed by a content and email refresh which was aimed at

  1. Generating directed content for each of these target groups.
  2. Presenting the benefits of the EPP program consistent with the tone and voice of Samsung.com.


The old flow did not provide a very seamless experience. The user had to search for an OPT in their email and then go back to Samsung.com where he had to enter the OPT to get access to offers. When the user exited the current session, he had to log back into the website using a default password that he had received via email. The user could then change the password on the website.
I worked on a new user flow and content strategy for this program. This simplified the number of steps the user had to go to access the offers. It simply sent the user a verification link that allowed the user to change his password. The user was then signed in and directed to the Offers Page.
